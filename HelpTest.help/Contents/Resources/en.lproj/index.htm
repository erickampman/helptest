MIDISource Help

<> functions in these fundamental ways:

  • Software MIDI keyboard - Latched and Unlatched modes
  • Arpeggiator
  • Metronome

The metronome is independent of the keyboard and arpeggiator so you can use it by itself or in combination with the other functions.

General Options
Mode

The mode sets the way in which the soft keyboard will be used.

  • Real Time
    • MIDI On/Off events are sent when the soft keyboard keys are clicked and released. 'Glissando' also supported.
  • Latched Real Time
    • Same as Real Time, but each key is "click-on', 'click-off'.
  • Arpeggiated
    • Each note, or group of notes, are played sequentially, at the 'Tempo' set in TIme Options, with the Note Value selection in the Arpeggiation Options tab.

Max Velocity

MIDI Note On Velocity (aka 'how hard you hit the key') ranges from 0 to 127. This slider sets the maximum velocity that will be transmitted. It is similar to a volume control, but many synthesizers/patches use the velocity control for other things in addition to volume, like filter cutoff frequency.

Output Channel

MIDI has 16 channels. Some synths ignore the channel; some can be configured to ignore the channel. DAWs sometimes provide no way to filter on MIDI channel.

Time Options
Tempo

The Tempo is in beats-per-minute. In 6/8 time one beat equals 3 eighth notes. I know, it's a bit odd, but in order to keep the UI simple I chose to do it this way.

Beats Per Measure

In x/4 this is the number of quarter notes in the measure. In x/8 this is the number of eighth notes per measure.

Denominator

Here you select between a quarter note getting 1 beat, x/4, or an eighth note getting 1 beat, x/8. The metronome is set to click every three beats in x/8 time. As a consequence, the Tempo is calculated in dotted quarters per beat, since in most cases x/8 time signatures have a triplet feel.

Arpeggiation Options

The Arpeggiation Options Tab supplies the Arpeggiation-specific controls. Other sections also affect arpeggiation as well, such as the Tempo in the Time Options Tab.

Note Value

You can choose among Sixteenth, Eighth, Quarter, and Whole notes. Note that a Whole note extends for an entire measure; it is not necessarily 4 beats.

Type
There are several supported Arpeggiation Types. Some are standard; some are unique to this app:
  • Up/Down
    The selected keys are played from bottom to top, and then back down, and then the cycle repeats from the bottom again.
  • Up
    The selected keys are played from bottom to top, and then the cycle repeats from the bottom again.
  • Down
    The selected keys are played from top to bottom, and then the cycle repeats from the top again.
  • Random
    Which of the selected keys are played is chosen randomly.
  • Random Walk
    The next note will either be the next one in the same direction (up or down), or will change direction (less likely), or will repeat the note (least likely).
  • Oopmah
    The bottom note will be played on the downbeat. The other notes will be played if the Note Value selection is other than Whole (note).
  • Oompah2
    The bottom note will be played on the downbeat. If the Note Value selection allows for it the second lowest note will played on beats other than the downbeat, and the rest of the notes will be played on offbeats.
Legato

This slider chooses how long each note will last.

Downbeat Velocity

This chooses the relative level of downbeat notes. It is in serial with the Max Velocity so will not override that value.

Beat Velocity

This chooses the relative level of downbeat notes.

Offbeat Velocity

This chooses the relative level of offbeat notes.

Metronome Options

The metronome lives in its own universe. You can enable/disable it independently of other functions.

Enabled

And here's how you do that.

Downbeat Volume

You can control whether the downbeat volume is louder than the beat volume.

Beat Volume

In some cases you might want to set the Beat Volume to 0, so only the downbeat gets a metronome.

Sound

You can choose a sound. This may inflict less psychological damage from hearing the same sound over and over again. I'm not sure.
